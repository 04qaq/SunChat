# 六、情绪系统（核心亮点）

---

## 6.1 情绪建模

### 定义：

```text
E_t ∈ [-1, 1]
```

---

## 6.2 更新公式

E_t = E_{t-1} \cdot \gamma + S_{new} \cdot (1-\gamma)

---

## ✔ 含义

| 项     | 含义     |
| ----- | ------ |
| E_t   | 当前情绪   |
| γ     | 记忆强度   |
| S_new | 当前输入情绪 |

---

## ✔ 代码实现

```python
self.value = self.value * gamma + s_new * (1 - gamma)
```

---

## ✔ 本质

👉 一个**指数加权移动平均（EMA）**

---

## ✔ 行为特点

| 输入   | 结果    |
| ---- | ----- |
| 连续正面 | 越来越开心 |
| 单次负面 | 不会突变  |
| 无输入  | 渐渐平稳  |
